Chapter 6 : Web Technology

Introduction to the Internet

Internet is the global network of networks. Or it is a huge collection of computers all over the world that are inter-connected to one another in various ways to form a web-net.

Components required for Internet Connections:

  • A TCP/IP (Transmission Control Protocol/Internet Protocol) enabled computer with web browser.
  • An account with an ISP (Internet Service Provider).
  • A telephone line.
  • A MODEM to connect to the telephone line and computer.
  • A set of computer with the following capacity:
    • Minimum 386 microprocessor chip with 16 MB RAM.
    • Color monitor with 256 color support and 640x480 resolution.
    • A hard disk with 200 MB of free space.
    • Multimedia sound card and speaker.

Internet Services and Applications

  • World Wide Web (WWW): WWW is a series of services that are interconnected through hypertext. It was created in 1990 by the European Particle Physics Laboratory in Geneva, Switzerland, to exchange and share data through the Internet using the protocol known as Hypertext Transfer Protocol (HTTP).
  • Web Browser: A web browser is a computer program that accesses web pages and displays them on a computer screen. Popular web browsers include Internet Explorer, Safari, Mozilla Firefox, Opera, and others.
  • FTP - File Transfer Protocol: FTP is a tool that permits an Internet user to move or transfer a file from one computer to another. The files may contain data, graphics, text, etc.
  • Electronic Mail (E-mail): E-mail is the most widely used feature on the Internet for sending and receiving messages electronically, which can include a variety of digital text, graphics, video, animation, and audio data.
  • Voicemail: Voicemail is a service offered by specialized hardware and software that answers telephone calls and records audio messages.
  • USENET - View and News: USENET is the Bulletin Board Service (BBS) of the Internet, where messages are organized into thousands of newsgroups covering specific areas.
  • Chat: Chat is a service that allows real-time communication by typing. It is a quick and easy way to exchange information and ideas, with offline messages available if the person is not online.
  • IRC - Internet Relay Chat: IRC is a multi-user live chat facility. Major IRC servers around the world are linked, and anyone can create a channel to chat with others.
  • Video Conferencing: Video conferencing, also known as webcam chatting, enables direct face-to-face communication across networks and provides real-time visual communication.
  • Universal Resource Locator (URL): A URL is a unique address that identifies the location of a resource on the Internet. It consists of the protocol (http), host computer name (www), domain name (animalplanet), and domain type (com).
  • E-Fax (Electronic Fax): E-Fax allows sending and receiving faxes directly on a computer. It functions similarly to E-mail, where messages go to an Internet server and are then forwarded to the destination.

Introduction to HTML

HTML stands for HyperText Markup Language, which is used to create web pages for the internet. HTML files are text files that contain several tags and attributes. It was developed by Tim Berners-Lee, based on SGML (Standard Generalized Markup Language). HTML uses these tags and attributes to organize and display content on a web browser. The versions of HTML include HTML, HTML+, HTML 1.0, HTML 2.0, HTML 3.2, HTML 4.01, and HTML 5.0.

Use of HTML:

  • It is used to create the basic layout or design of a web page.
  • Without HTML, the worldwide web wouldn't exist.
  • It allows embedding images, videos, audio, etc.
  • It enables customization and formatting of characters, such as bold, italic, and underline.
  • It helps create links and lists.

Advantages of HTML:

  • It is easy to use, flexible, and user-friendly.
  • It is supported by all web browsers and operating systems like Macintosh, Windows, Unix, etc.
  • It is easy to obtain and modify content.
  • It is compatible with all search engines.
  • HTML 5.0 introduces several new and advanced features.
  • HTML files load quickly.
  • Many tutorials are available for learning HTML.

Disadvantages of HTML:

  • It is difficult to create attractive and interactive web pages only with HTML. Other technologies like PHP, JavaScript, and SQL are often required.
  • It cannot create dynamic web pages.
  • It is a markup language, not a programming language.
  • There are many inconsistencies and compatibility issues with HTML.
  • There is no fully accepted standard for HTML.

HTML Tags

HTML tags are instructions or commands given to the web browser to organize text, images, and other components of web pages. These tags are executed by the web browser to render content. Tags are written inside angular brackets <>. There are two types of HTML tags: opening tags and closing tags.

DBMS (Database Management System)

Data: Data refers to raw facts and figures. They are isolated and uninterpreted until processed. For example, Ram, 19, 12, Pokhara.

Information: Information is the meaningful result obtained after processing data. For example, "Ram is a student, 19 years old, who lives in Pokhara-12."

Databases are organized collections of data, designed to store and process information quickly and efficiently. A Database Management System (DBMS) helps manage this data. Examples include MS Access, MySQL, Oracle, and dBase.

Advantages/Features/Importance of DBMS:

  • Quick and easy data retrieval.
  • Reduces data redundancy.
  • Allows data sharing over a network.
  • Improves data security and privacy.
  • Helps reduce data inconsistency.
  • Provides an organized way to store data.

Disadvantages of DBMS:

  • Higher installation and operating costs.
  • Requires more disk space.
  • Potential security and privacy issues.
  • Requires technical manpower (DBA) for maintenance and operation.
