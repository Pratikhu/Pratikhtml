Iterative Control Structure

For

The for loops are iterated over a range of values or elements in a collection. These are mainly used where the number of iterations is known beforehand like iterating through elements in an array or predefined range.

Syntax: for (initialization; condition; update) { * code to be executed * }
Example Program:
To print pattern in C
#include <stdio>>
int main()
{
for (int i = 0; i < 5; i++) {
printf("%d ", i);
}
printf("\n");
int count = 0;
while (count < 5) {
printf("%d ", count);
count++;
}
printf("\n");
count = 0;
do {
printf("%d ", count);
count++;
} while (count < 5);
printf("\n");
return 0;
}

OUTPUT:

While

While loops execute as long as specifies condition evaluates to true. These are suitable for situations where the number of iterations is not known and depends on dynamic conditions.

Syntax: while (condition) { * code to be executed * }
Example Program:
To print sum of first 10 natural numbers
#include <stdio.h>>
int main() {
int sum = 0, i = 1;
while (i <= 10) {

sum += i;
i++;
}
printf("%d", sum);
return 0;
}

OUTPUT:

Do While

Do while loops are similar to while loop but guarantee at least one execution of the loop body before checking the condition. These are useful when loop must execute at least once regardless of the condition.

Syntax: do { * code to be executed * } while (condition);
Example Program:
To print a message multiple times
#include <stdio.h>
int main(){
int a = 1;
do{
printf("Hello World\n");
a++;
} while(a <= 5);
printf("End of loop");
return 0;
}

Arrays in C

Arrays are used to store multiple values in a single variable, instead of declaring separate variables for each value. To create an array, define the data type (like int) and specify the name of the array followed by square brackets []. To insert values to it, use a comma-separated list inside curly braces, and make sure all values are of the same data type:

Mainly of two types:

1-Dimensional Array

A one-dimensional array can be viewed as a linear sequence of elements. We can only increase or decrease its size in a single direction.

Syntax:

element_type array_name[array_size];

2-Dimensional Array

A 2D array is also known as a matrix (a table of rows and columns).

Syntax:

element_type array_name[no.of rows][no. of columns];

String Array

In C, an array of strings is a 2D array where each row contains a sequence of characters terminated by a ‘\0’ Null character (strings). It is used to store multiple strings in a single array.

Syntax:

char arr_name[r][m] = {s1, s2, …., sn};

Example Program:
To print a message multiple times

#include <stdio.h>
int main() {
char arr[5][15] = {"Pratik", "Pratiks", "Bajracharya"};
for (int i = 0; i < 3; i++)
printf("%s\n", arr[i]);
return 0;
}
